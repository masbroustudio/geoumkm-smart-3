📋 Ringkasan Proyek

GeoUMKM Intelligence 3.0 adalah sistem intelijen lokasi berbasis AI dan geospatial analytics yang membantu UMKM memilih lokasi usaha optimal dan membantu pemerintah mengidentifikasi area prioritas intervensi di Jawa Barat. Proyek ini menggunakan 6.000 data UMKM sintetis dengan 18+ fitur advance dan analisis statistik mendalam.

📓 Notebook Documentation

01. Data Generation v3

Generate 6.000 data UMKM Jawa Barat dengan skor potensi realistis (45-95) dan 18+ fitur advance.

02. Clustering & Recommendation

K-Means clustering (8 cluster), Top 15 rekomendasi lokasi, What-If Analysis, dan Government Priority Dashboard.

03. Final Summary

Ringkasan statistik, estimasi dampak, dan kesimpulan proyek untuk portofolio.

04. Visualisasi Distribusi Skor

Histogram, box plot, violin plot, dan visualisasi distribusi skor potensi.

05. Analisis Statistik Mendalam

Kruskal-Wallis, Tukey HSD, regresi linear berganda, effect size, dan uji normalitas.

06. Visualisasi Lengkap

Q-Q Plot, CDF, heatmap korelasi, dan visualisasi statistik lanjutan.

✨ Fitur Utama

  • ✅ Advanced Geospatial Feature Engineering — 18+ fitur (jarak ke pusat ekonomi, skor infrastruktur, cluster success rate, dll)
  • ✅ K-Means Clustering — 8 cluster dengan karakteristik unik untuk segmentasi wilayah
  • ✅ Location Recommendation Engine — Top 15 lokasi terbaik per jenis usaha
  • ✅ What-If Analysis Simulator — Prediksi skor potensi untuk skenario lokasi baru
  • ✅ Government Priority Dashboard — 15 kecamatan dengan skor terendah + rekomendasi intervensi
  • ✅ Analisis Statistik Mendalam — Kruskal-Wallis, Tukey HSD, regresi linear, effect size

🛠️ Technology Stack

Azure Synapse Analytics + Python (XGBoost, Folium, Seaborn, SciPy, Statsmodels) + Apache Spark + Azure Static Web Apps
